# ☄️ NASA NEO — Аналіз навколоземних астероїдів

Веб-застосунок для отримання, обробки та візуалізації даних про астероїди,
що наближаються до Землі, на основі відкритого **NASA Near Earth Object Web Service**.

---

## Мета проєкту

Розробити інтерактивний Streamlit-застосунок, що дозволяє:
- завантажувати актуальні дані про навколоземні астероїди з NASA API;
- зберігати їх у форматі CSV;
- аналізувати й візуалізувати ключові характеристики астероїдів.

---

## Джерело даних

**NASA Near Earth Object Web Service (NeoWs)**  
API endpoint: `https://api.nasa.gov/neo/rest/v1/feed`  
Документація: https://api.nasa.gov  
API-ключ для тестування: `DEMO_KEY`

> **Обмеження API:** один запит охоплює не більше **7 днів**.  
> При виборі більшого діапазону застосунок автоматично розбиває запит на частини.

---

## Структура даних

Після обробки формується датафрейм із такими полями:

| Поле              | Тип     | Опис                                      |
|-------------------|---------|-------------------------------------------|
| `name`            | str     | Назва астероїда                           |
| `date`            | date    | Дата зближення із Землею                  |
| `is_hazardous`    | bool    | Потенційно небезпечний (True/False)       |
| `diameter_min_m`  | float   | Мінімальний діаметр (метри)               |
| `diameter_max_m`  | float   | Максимальний діаметр (метри)              |
| `diameter_avg_m`  | float   | Середній діаметр: (min + max) / 2         |
| `velocity_kmh`    | float   | Відносна швидкість (км/год)               |
| `miss_distance_km`| float   | Відстань до Землі (км)                    |
| `orbiting_body`   | str     | Тіло, навколо якого пролітає (Earth тощо) |

---

## Основні можливості застосунку

- **Вибір часового діапазону:**
  - попередні 7 днів
  - наступні 7 днів
  - довільний діапазон (з автоматичним поділом на 7-денні блоки)

- **Зведена статистика (KPI-метрики):**
  - загальна кількість астероїдів
  - кількість потенційно небезпечних
  - найближчий, найбільший та найшвидший астероїди

- **Інтерактивна таблиця** з фільтрацією за:
  - типом (небезпечний / безпечний)
  - мінімальним діаметром
  - максимальною відстанню

- **Завантаження CSV** одним кліком

- **Детальна картка** конкретного астероїда з gauge-індикатором розміру

## Реалізовані графіки

| # | Назва | Опис |
|---|-------|------|
| 1 | **Астероїди за днями** | Стовпчаста діаграма кількості астероїдів (усього та небезпечних) за кожен день |
| 2 | **Співвідношення небезпечних** | Кругова діаграма (donut) безпечних і небезпечних астероїдів |
| 3 | **Топ-10 найближчих** | Горизонтальна барна діаграма 10 астероїдів з найменшою відстанню до Землі |
| 4 | **Розмір ↔ Відстань** | Точкова діаграма залежності середнього діаметра від відстані (розмір бульки = діаметр) |
| 5 | **Швидкість ↔ Відстань** | Точкова діаграма з лінією тренду (OLS) |

## Інструкція запуску

### 1. Клонуйте репозиторій
```bash
git clone https://github.com/<your-username>/nasa-neo-asteroids.git
cd nasa-neo-asteroids
```

### 2. Створіть та активуйте віртуальне середовище
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# macOS / Linux
source venv/bin/activate
```

### 3. Встановіть залежності
```bash
pip install -r requirements.txt
```

### 4. Запустіть застосунок
```bash
streamlit run app.py
```

Застосунок відкриється у браузері за адресою `http://localhost:8501`.
